Component Breakdown

Presentation Tier (Frontend UI)

A lightweight, dependency-free interface built using Vanilla HTML, CSS, and JavaScript. It serves the Live Dashboard, Dataset Explorer, and Explainable AI (XAI) visualizer, providing real-time interactions without the overhead of heavy frameworks.

Application Tier (FastAPI Backend)

A robust Python-based API layer operating on the FastAPI framework. It acts as the central gateway, securely routing HTTP REST API requests from the frontend client to the appropriate backend microservices.

Service Modules

The business logic is decoupled into three main services: The Dataset Service handles incoming network flow data parsing; the Threat Intel Service manages external Indicator of Compromise (IOC) feeds; and the Prediction Engine pre-processes data windows for inference.

AI & Machine Learning Engine

Sequential Intelligence (Temporal Modeling): The architecture employs LSTM units to capture long-range temporal dependencies. Ensemble Baseline (Predictive Power): High-speed, high-performance classification is driven by an ensemble of XGBoost and LightGBM. This combination optimizes inference speed while maximizing predictive accuracy across diverse threat datasets.

Data Persistence Tier

For development and initial lab testing, SQLite is utilized for rapid prototyping. For production deployment, the system is designed to interface with PostgreSQL to handle large volumes of high-velocity network flow logs and prediction histories reliably.
